# EDA_and_Rfundamentals
The goal was to perform Exploratory Data Analysis(EDA) of a given dataset and apply new knowledge on R fundamentals.


Given the data obtained from the *"Behavioral Risk Factor Surveillance System BRFSS survey"* (https://www.kaggle.com/datasets/alexteboul/heart-disease-health-indicators-dataset/data), this project intended to investigate the association between age, sex, socioeconomic and educational status, diet and physical activity with obesity in
the study population. As we were starting to delve into how to use R for statistical purposes, this project was based entirely on descriptive and exploratory analysis.

Please check the R script(*grupo3_script_relatorio_LE.R*) as well as the resulting written report(*relatorio_grupo3_LE.pdf*). If you wish to run the script, you'll need to download the dataset (*heart_disease_health_indicators_BRFSS2015.csv*).
